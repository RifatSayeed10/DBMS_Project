# **🚀 Dynamic Features**

Explore a powerful collection of features designed to deliver a seamless research experience:

* **📚 Unified Scholarly Catalog** — Search and browse academic resources from a centralized interface.
* **🔖 Personal Bookmarks** — Save and organize favorite papers for quick access.
* **👤 Interactive Researcher Profiles** — View detailed author information, publications, and related works.
* **📄 Self-Healing PDF Delivery Engine** — Automatically detects unavailable files and retrieves backup sources to ensure reliable PDF access.

# **🛠️ Technology Stack**

Built with a modern, scalable, and performance-focused architecture.

* **React 19** for building fast, interactive user interfaces.

### **Backend**

# **⚡ Getting Started**

Follow these simple steps to run the project locally.

### **1. Clone the Repository**

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```

### **2. Install Dependencies**

```bash
npm install
```

### **3. Configure Environment Variables**

Create a **`.env`** file in the project root and add the required environment variables.

```env
PORT=5000
DATABASE_URL=your_database_url
API_KEY=your_api_key
```

### **4. Start the Development Server**

```bash
npm run dev
```

### **5. Build for Production**

```bash
npm run build
npm run start
```
