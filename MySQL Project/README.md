# 💰 Expense Tracker

A simple **Expense Tracker** application built with **MySQL** that helps users manage their daily income and expenses. The project allows users to store financial records, monitor transactions, and keep track of their overall balance efficiently.

## ✨ Features

* ➕ Add income and expense records
* ✏️ Update existing transactions
* ❌ Delete transactions
* 📋 View all financial records
* 💾 MySQL database integration

## 🛠️ Tech Stack

* MySQL
* SQL

## 🚀 Getting Started

1. Clone the repository.
2. Import the SQL database file into MySQL.
3. Configure your database connection.
4. Run the SQL queries or application.
